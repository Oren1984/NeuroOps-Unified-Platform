# NeuroOps – Static Demo

This folder contains a fully static demo version of the **NeuroOps Unified Platform**.

## Overview

The static demo is designed for **presentation and portfolio purposes only**.
It replicates the UI, navigation, and experience of the platform without requiring any backend, Docker services, or external dependencies.

## Key Features

* Full UI simulation of the NeuroOps platform
* Navigation between all modules via sidebar
* Static mock data (no API calls)
* Zero dependencies (no build step required)
* Works on any static hosting environment

## Included Pages

* Dashboard
* Control Room
* Live Control
* Incident Replay
* Career Agent
* Insight Engine
* Warehouse Copilot
* Platform Events
* Platform Overview

## How to Run Locally

### Option – Open directly

Open the file in your browser:

```
static-demo/index.html
```

---

## Deployment

This demo can be deployed to any static hosting platform:

* IIS
* GitHub Pages
* Netlify
* Vercel
* AWS S3

Simply upload the contents of this folder:

```
index.html
css/
js/
```

No build or installation is required.

## Notes

* This is a **demo-only version** – backend functionality is not included
* All data is static and pre-defined
* Embedded services (Career Agent, Insight Engine, etc.) are simulated

## Purpose

This static demo is intended to:

* Showcase the platform UI and architecture
* Provide a fast and stable presentation experience
* Enable easy sharing without infrastructure setup
